List Element

List Element is used to create a list of item or make a list of given items. There are Two types of list Element as following:-

  • Un-Ordered List

    • An unordered list is used to create a list where it is not mandatory to maintain an order of the items.

  • Ordered List

    • An Ordered List is used to create list of elements where the order or hierarchy of required to be maintained.

Anchor Element

An Anchor element in HTML represents a hyperlink, which makes it possible for users to click on a web page and be directed. There are Two Types of URLs:-

  1. Absolute URL

    2. Absolute URLs are used to an absolute web page in which user can go to the specific web page directly by just single click. we use <a href="URL">text</a> syntax to create a hyperlink.

  2. Relative URL

    3. Relative URLs are used to visit diferent pages located with respect to the current file.
